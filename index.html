Cooldown Chess


Created a real-time multiplayer web-game where users play chess without turns using React and Node.js. Instead of being turn-based, players will be given a "cooldown" after moving that they must wait before they can move again. The project uses websockets with Socket.IO to enable the bidirectional flow of data between the client and server


Created with: React, Node.js, Express.js, socket.io

OnBoard


Created a Learning Management System so that onboarding courses can be created and edited for new employees. Course creators can add text, images, videos, and tasks to their courses while in edit mode, and a real-time leaderboard incentivizes employees to complete lessons.


Created with: React, Node.js, Express.js, MongoDB, Mongoose, socket.io

StatBook


A data analytics dashboard for Facebook users. The dashboard presents data such as top engagers, a summary of reactions, and suggestions. Suggestions are made with a recommendation engine that analyzes engagement of previous posts and extracts key words.


Created with: React, Node.js, Express.js, Facebook Graph API, Microsoft Azure

Golf Tournament AR


iOS app which incorporates augmented reality into the golf tournament spectating experience. The app displays the players and hole locations around, which can be clicked on to receive additional stats. A simple Android app was built to send real-time GPS coordinates to a Flask server so that the main app can capture player movement.


Created with: Swift, ARKit, Python, Flask, AWS, Kotlin

Forest Flips

3D physics-based driving game developed in Unity. The player needs to score as many points as possible by doing stunts such as flips and rail slides. Points for a flip are determined by the angle the car has rotated mid-air and the horizontal speed of the car. Not properly landing a stunt results in losing the points for it, so precision is key.


Created with: Unity Engine, C#
