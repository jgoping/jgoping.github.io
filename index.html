Projects ▼

OrigamIT

Android app created for Hack The North 2017. Given a picture of something through either the camera or in storage, the app will identify what is in the picture using the Watson image recognition API. Once the subject of the photo is identified, this is run through an origami-instruction database to return the user with instructions of it.


Created with: Android Studio, IBM Watson API


Tasks.Show()

For my first project using Node.js, decided to create a website to keep track of my ongoing to-do list. Tasks are stored within a MongoDB database, where they can be added, removed, and altered. The tasks can be sorted alphabetically, by due date, or by priority by the sort buttons, and clicking the sort button again will reverse the order.


Created with: Node.js, Express.js, MongoDB, jQuery, AJAX, HTML, CSS, JavaScript

Lost & Found

Android app created for Hack Western IV. Due to the high amount of people who lose things in University, the app serves as a platform dedicated solely to helping people find their things. Used Watson image recognition API to help automate the uploading process and firebase to store the uploaded items.


Created with: Android Studio, IBM Watson API, Firebase

The Search For DK

First-person Python game with a map system based on an undirected graph where the player walks around my basement looking for a randomly hidden Donkey Kong plush toy! Made it with a friend with little programming experience as a fun way to further interest them into the field of Software.



Created with: Python + Pygame library


WikiNotes

Website created with team for Tech Under 20 Cup. This website allows students to upload, view, and edit notes that they have taken so knowledge can get passed down after a course ends. Worked on making the notes editable using Quilljs textboxes, familiarizing myself more with HTML and jQuery along the way.


Created with: HTML, CSS, Javascript, jQuery, Quilljs

Turing Hero

Inspired by a game online called 'Flash Hero', created a rhythm game for a school Computer Science Project which tries to emulate playing guitar on your keyboard.
