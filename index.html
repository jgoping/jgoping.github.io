Projects ▼

OrigamIt

Android app created for Hack The North 2017. Given a picture of something through either the camera or in storage, the app will identify what is in the picture using the Watson image recognition API. Once the subject of the photo is identified, this is run through an origami-instruction database to return the user with instructions of it.

Created with: Android Studio, IBM Watson API

Lost & Found

Android app created for Hack Western IV. Due to the high amount of people who lose things in University, the app serves as a platform dedicated solely to helping people find their things. Used Watson image recognition API to help automate the uploading process.

Created with: Android Studio, IBM Watson API, Firebase
